# snakegame 


1- overview of game-
This is a basic Snake Game made using C language. The player controls a snake that moves around the screen and eats food. Every time the snake eats food, it grows longer. The goal is to eat as much food as possible without hitting the walls or the snake’s own body.


2- Rules
The snake dies if it hits the wall.
The snake dies if it bites itself.
Each food eaten increases the score and the snake’s length.
The game ends when the snake dies.


3- Controls
- W → Move Up
- S → Move Down
- A → Move Left
- D → Move Right
- X → Exit the game
- and i have also added arrow keys for movement.

  
4- Features
Score tracking
Snake grows after eating food
Game over screen
Simple console graphics using characters
